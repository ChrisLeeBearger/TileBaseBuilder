## About

TileBaseBuilder is a 2D base building game created with [Unity](https://unity.com/) and is heavily inspired by games like [RimWorld](https://store.steampowered.com/app/294100/RimWorld/).

## Project Goals

- [x] Wall placement system
- [x] Job queue
- [ ] Resource managment
- [ ] Pathfinding
- [ ] Combat system

## Live Demo
